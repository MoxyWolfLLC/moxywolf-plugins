"[START WITH MOST IMPACTFUL DATA POINT OR DATE]"

[EXPLAIN THE CONNECTION THAT'S BEEN HIDING IN PLAIN SIGHT]

→ [STAT 1 with source]
→ [STAT 2 with source]
→ [STAT 3 with source]

Host 1: "Let's walk through the math on why manual approaches can't scale..."

Example: "So if you have 110 controls, and they update every 90 days, and each validation takes X hours..."

Host 2 pushes back: "But what about organizations that already have NIST 800-171 compliance?"

Host 1 addresses: "[HANDLE THE OBJECTION WITH DATA]"

Discuss the strategic concept:
[EXPLAIN THE MULTIPLIER EFFECT / KEY FRAMEWORK]

Host 2: "Here's what I'm seeing in practice..."
[SHARE REAL-WORLD EXAMPLES WITH SPECIFIC OUTCOMES]

**ACT 3: THE STRATEGIC PLAY (7-10 min)**

Host 1: "Let's talk timelines. Here are the dates that matter..."

Timeline discussion:
- [DATE 1]: [WHAT HAPPENED]
- [DATE 2]: [WHAT'S COMING]
- [DATE 3]: [FINAL DEADLINE]

Host 2: "So who wins in this environment?"

- [WHO BENEFITS AND WHY]

[COMPETITIVE ADVANTAGE REFRAME]

Closing:
Host 2: "So what should listeners do immediately?"

"Read the complete analysis on the STIGViewer blog"

### Example 1: CMMC/STIG Convergence Video Prompt

```markdown
**Context:**
You are discussing the convergence of DISA STIGs and CMMC 2.0 enforcement following November 10, 2025 Phase 1 activation.

**Pronunciation Guide:**
- STIG = "stig"
- DISA = "dissah"
- CMMC = "C-M-M-C"

**ACT 1: THE WAKE-UP CALL (0-3 min)**

Host 1: "Three months ago, CMMC 2.0 was theory. Today, it's enforceable law affecting 338,000 defense contractors."

Host 2: "And here's what most contractors are discovering too late—the secret to passing isn't in the CMMC framework itself."

Host 1: "Right. It's been hiding in plain sight for 20 years inside DISA STIGs. Why didn't we see this coming?"

Host 2: "Because most DIB contractors treated STIGs as internal DoD requirements, not something that applied to them. But now that's changed."

**Brutal Reality Data Points:**
→ STIGs update every 90 days (January 2025 release just dropped)
→ 110 NIST 800-171 controls must be continuously validated for Level 2
→ Manual compliance costs averaging $51,095.60 with $41,666 recurring annually
→ Phase 2 (Nov 2026) eliminates self-assessment safety net

**ACT 2: THE TECHNICAL REALITY (3-7 min)**

Host 1: "Let's walk through why manual STIG compliance doesn't scale. You've got how many controls?"

Host 2: "For Level 2, 110 NIST 800-171 controls. But here's the kicker—DISA updates STIGs quarterly."

Host 1: "So every 90 days, you're potentially re-validating configurations across your entire environment?"

Host 2: "Exactly. And most organizations have what, 50-200 systems? Do that math manually."

Host 1: "That's why we're seeing 70-90% cost reduction with automation. It's not optional anymore."

**Debate Moment:**
Host 2: "But what about organizations that already have NIST 800-171 compliance?"

Host 1: "Documentation isn't technical proof. Auditors with DoD backgrounds expect STIG-level detail. You can say you follow best practices, or you can show your Windows servers meet Windows Server STIG baselines."

**The Multiplier Effect:**
Host 2: "This is where it gets interesting—STIG compliance isn't just about CMMC."

Host 1: "Right. Because STIGs link to NIST 800-53, which is the parent for 800-171, you're simultaneously strengthening your position for FedRAMP, ISO 27001, SOC 2..."

Host 2: "It's a compliance multiplier. Do the hardening once correctly, and you satisfy multiple frameworks."

**ACT 3: THE STRATEGIC PLAY (7-10 min)**

Host 1: "Timeline check. Where are we?"

Timeline:
- Nov 10, 2025: Phase 1 active (we're here—2 months in)
- Nov 10, 2026: Phase 2 requires third-party assessments (10 months away)
- Most orgs need 6-12 months to prepare

Host 2: "So if you're responding to an RFP with Q3 2026 award date, your runway is..."

Host 1: "Already tight. And don't forget—certification must be valid at contract award."

**Competitive Dynamics:**
Host 2: "Who wins here?"

Host 1: "Organizations that automated early. They're building genuine security postures while competitors scramble. And they've turned compliance into a competitive moat."

Host 2: "What about the 68% who are small businesses?"

Host 1: "That's where we'll see consolidation. Larger primes will require supplier certification. Some small contractors won't make it."

**Reframe:**
Host 2: "So this isn't just a burden—it's actually a barrier to entry for your competition."

Host 1: "Exactly. If you're prepared, you're winning contracts while others can't even bid."

**Closing:**
Host 2: "What should listeners do immediately?"

Host 1: "Start with the 90-day implementation checklist. Phase 1 is assessment—identify which STIGs apply, run baseline scans. Phase 2 is hardening—implement automation. Phase 3 is continuous compliance."

**Key Quotes:**
1. "STIGs aren't bureaucratic overhead—they're the granular configurations that directly support NIST 800-171 controls."
2. "Organizations using STIG automation are reducing costs by 70-90% and building genuine security postures that auditors trust."
3. "This isn't compliance theater. It's operational security that happens to satisfy auditors."

**CTA:**
"Download the free 90-day CMMC/STIG implementation checklist at STIGViewer.com"
```

## Audio Content Templates (1.5 Minutes)

### Template: Ultra-Condensed Audio Prompt

```markdown
**Format:** NotebookLM audio-only, 90 seconds maximum

**Pronunciation:** STIG = "stig", DISA = "dissah"

**Target:** Commuters, people doing quick research, time-constrained professionals

**Structure:**

**OPENING (0-15 sec) - The Hook:**
[Single most important stat or development]

Example: "On November 10, 2025, the DoD made CMMC 2.0 mandatory for 338,000 defense contractors. Most are discovering the secret to passing audits too late."

**CORE INSIGHT (15-60 sec) - The One Thing:**
[The revelation that changes everything]

Example: "The secret? DISA STIGs. Security Technical Implementation Guides—they're the granular configurations that directly support NIST 800-171 controls. But STIGs update every 90 days. Manual compliance? Dead. Organizations using automation are cutting costs by 70-90% and building security postures auditors actually trust."

**BRUTAL REALITY (60-75 sec) - The Urgency:**
[Why this matters RIGHT NOW with 2-3 data points]

Example: "Here's the timeline. Phase 1 enforcement started November 10th. Phase 2—requiring third-party assessments—hits November 2026. Most organizations need 6-12 months to prepare. If you're bidding on Q3 2026 contracts, your runway is already tight."

**ACTION (75-85 sec) - The Next Step:**
[What to do immediately]

Example: "Start with the 90-day implementation checklist. Days 1-30: assessment. Days 31-60: automation. Days 61-90: continuous compliance."

**CLOSING (85-90 sec) - The Memorable Line:**
[Quote or reframe that sticks]

Example: "This isn't just compliance. It's operational security that happens to satisfy auditors. And right now, it's a competitive advantage."

**CTA:**
"Full analysis and free checklist at STIGViewer.com"

**Tone Notes:**
- No filler words or phrases
- Every sentence earns its place
- Use verbal signposting ("Here's the critical stat..." "Three things matter...")
- Assume listener can't reference visuals
- Data-driven, urgent, actionable
```

## Social Media Clip Guidelines

### 30-Second Stat Clip

**Purpose:** Shock and engage with data point

**Format:** Square (1080×1080px)

**Structure:**
- 0-3 sec: Hook visual (number or date appearing)
- 3-15 sec: Context with burned-in captions
- 15-25 sec: "Why this matters" with supporting visual
- 25-30 sec: CTA "Full analysis in comments ↓"

**Caption Strategy:**
"Fire Orange text over Electric Blue background"
Aeonik Bold for numbers/dates
Aeonik Regular for supporting text

**Example:**
```
0-3s: "338,000" appears in Fire Orange
3-10s: "Defense contractors must achieve CMMC compliance"
10-20s: "68% are small businesses, most unprepared"
20-25s: Graph showing cost comparison
25-30s: "Download the free implementation guide ↓"
```

### 45-60 Second Insight Clip

**Purpose:** Share the "aha" moment from video

**Format:** Horizontal (1920×1080px) or Square

**Structure:**
- 0-5 sec: Setup question
- 5-35 sec: The insight/revelation
- 35-50 sec: Supporting example or data
- 50-60 sec: CTA to full video

**Visual Elements:**
- B-roll relevant to topic
- Key quotes in text overlays (Fire Orange)
- Transitions on beat with dialogue

**Example:**
"Why STIGs Matter for CMMC Audits"
- Hosts discussing the hidden dependency
- Visual showing STIG → 800-53 → 800-171 → CMMC chain
- "This changes everything" moment
- "Watch full video at link in bio"

### 30-Second Vertical Action Clip

**Purpose:** Direct call to action for mobile users

**Format:** Vertical (1080×1920px) for Stories/Reels/Shorts

**Structure:**
- 0-3 sec: Attention-grabbing visual
- 3-20 sec: Single action step explained
- 20-27 sec: Why it matters NOW
- 27-30 sec: Swipe up/link in bio CTA

**Design:**
- Large text fills vertical space
- Fire Orange for action items
- High contrast for mobile viewing
- Minimal visual complexity

**Example:**
"Start Your CMMC Compliance Today"
- "Step 1: Download the 90-day checklist"
- "Phase 1: Assessment (Days 1-30)"
- "Phase 2: Implementation (Days 31-60)"
- "Get started → Link in bio"

## Quality Checklist

**Video Content:**
- [ ] Three-act structure clearly defined
- [ ] Pronunciation guide included (STIG, DISA)
- [ ] Debate moment incorporated
- [ ] Emotional beats mapped
- [ ] Key quotes specified
- [ ] Analogies provided
- [ ] 5-10 minute target length maintained
- [ ] CTA clear and compelling

**Audio Content:**
- [ ] 90 seconds or less
- [ ] Hook in first 15 seconds
- [ ] One core insight emphasized
- [ ] Verbal signposting used
- [ ] No filler language
- [ ] CTA included
- [ ] Memorable closing line

**Infographic:**
- [ ] MoxyWolf color palette used correctly
- [ ] Fire Orange for headline and CTAs
- [ ] Aeonik typography (or fallback)
- [ ] Clear visual hierarchy
- [ ] 3-5 key data points
- [ ] Logo included but subtle
- [ ] Source attribution present
- [ ] QR code or URL for CTA
- [ ] Generous negative space
- [ ] Works at multiple sizes

**Video Thumbnail:**
- [ ] 1280×720px dimension
- [ ] High contrast colors
- [ ] Text readable at thumbnail size
- [ ] One dominant visual element
- [ ] Brand elements present
- [ ] Under 2MB file size

**Social Clips:**
- [ ] Multiple format versions (square, horizontal, vertical)
- [ ] Burned-in captions
- [ ] MoxyWolf color styling
- [ ] Hook in first 3 seconds
- [ ] Clear CTA
- [ ] High-quality audio

**Distribution:**
- [ ] Platform-specific posting times

- [ ] Cross-promotion language prepared
- [ ] Email sequence scheduled
- [ ] Engagement plan ready
- [ ] Analytics tracking configured
