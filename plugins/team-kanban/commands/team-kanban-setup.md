---
description: Set up the team kanban Canvas in Slack #general
---

Run the team-kanban skill in Mode 3 (Setup) to initialize the team kanban board.

This is a one-time setup that:
1. Finds the #general Slack channel
2. Creates the Team Kanban Canvas
3. Posts an introductory message explaining the board to the team
4. Stores the Canvas ID and channel ID in the Obsidian vault config
5. Offers to create a scheduled task for daily automated syncs

Read the team-kanban skill at `${CLAUDE_PLUGIN_ROOT}/skills/team-kanban/SKILL.md` and follow Mode 3 instructions.
